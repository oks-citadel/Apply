## Accessibility Standards

### Minimum Requirements
- ✅ Text contrast: **4.5:1** (normal), **3:1** (large 18px+)
- ✅ Touch targets: **44×44px** minimum
- ✅ Focus indicators: **2px solid Primary Blue**, 2px offset
- ✅ Body text: **16px minimum**
- ✅ Line height: **1.5× minimum** for body text
- ✅ Color + another indicator (never color alone)

### Testing Checklist
- [ ] Screen reader compatible (NVDA, JAWS, VoiceOver)
- [ ] Keyboard navigation functional
- [ ] Works at 200% zoom
- [ ] Passes automated tools (aXe, WAVE)
- [ ] Respects `prefers-reduced-motion`

## Common Mistakes to Avoid

### ❌ Typography
- Using random font sizes not in scale
- Line heights too tight
- Lines longer than 90 characters
- Poor contrast (gray on light gray)

### ❌ Colors
- Using colors not in palette
- Low contrast text
- Color as only indicator
- Too many colors competing

### ❌ Spacing
- Not using 4px multiples
- Inconsistent padding
- Cramped layouts
- Random spacing values

### ❌ Logo
- Stretching or distorting
- Too small to read
- On busy backgrounds
- Insufficient clear space
