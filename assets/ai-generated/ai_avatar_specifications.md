## Professional Avatars

### Overview
Modern 3D illustration style avatars representing diverse professionals across various industries and career stages. These avatars will be used in onboarding flows, testimonials, marketing materials, and UI placeholders.

### Technical Specifications
- **Format**: PNG with transparent background
- **Resolution**: 1024x1024px (high-res), 512x512px (standard), 256x256px (thumbnail)
- **Color Profile**: sRGB
- **Export Options**: WebP for web, PNG for print

### Style Guidelines
- **Art Style**: Modern 3D illustration, semi-realistic
- **Rendering**: Soft lighting, subtle shadows
- **Detail Level**: Clean, not overly detailed
- **Consistency**: Unified art style across all avatars
- **Accessibility**: High contrast, clear features

## AI Assistant Avatar

### Character Profile
- **Name**: Pilot (the ApplyforUs AI assistant)
- **Personality**: Helpful, smart, friendly, encouraging
- **Purpose**: Guide users, provide help, celebrate wins

### Design Specifications

#### Base Character
```
A friendly 3D robot/AI character with a modern, approachable design. Rounded body
shape with smooth surfaces, primarily in brand purple gradient (#8B5CF6 to #6366F1).
Expressive circular "face" screen that can display different emotions through simple
geometric patterns (dots for eyes, arc for mouth). Floating hands/arms that can
gesture. Small antenna or holographic projection element above head. Subtle glow
effects. Background: Transparent. Style: Modern, friendly tech aesthetic, not
intimidating. Full body shot, 1024x1024px.
```

#### Expression Variations

**1. Helpful (Default)**
```
Pilot AI assistant with welcoming expression (two bright dots for eyes, gentle
smile arc). One hand raised in greeting gesture. Soft purple glow. Conveying
"I'm here to help" energy.
```

**2. Thinking/Processing**
```
Pilot AI assistant with thoughtful expression (eyes slightly dimmed, horizontal
thinking line for mouth). One hand on chin area in contemplative pose. Animated
dots or loading indicator near head. Subtle pulsing glow.
```

**3. Celebrating/Success**
```
Pilot AI assistant with excited expression (large bright star-shaped eyes,
big smile arc). Both arms raised in celebration. Confetti or sparkle effects
around. Bright, vibrant glow. Conveying achievement and success.
```

**4. Suggesting/Recommending**
```
Pilot AI assistant with confident expression (bright focused eyes, slight smile).
One arm extended pointing or presenting. Holographic UI elements floating nearby
(job cards, checkmarks). Professional but friendly glow.
```

**5. Listening/Attentive**
```
Pilot AI assistant with attentive expression (large open eyes, neutral mouth).
Slight forward lean, hands clasped or one hand near "ear" area. Sound wave
visualizations nearby. Engaged, focused appearance.
```

**6. Error/Concerned**
```
Pilot AI assistant with concerned but not alarming expression (wide eyes with
slight tilt, small 'o' shaped mouth). One hand raised in "oops" gesture. Gentle
orange/yellow accent color mixed with purple. Empathetic, apologetic demeanor.
```

## Avatar Usage Guidelines

### Where to Use

#### Professional Avatars
- **Onboarding Flow**: Represent different user personas
- **Testimonials**: Placeholder for user reviews
- **Marketing Materials**: Hero sections, feature explanations
- **Empty States**: "No applications yet" screens
- **Loading States**: User profile placeholders
- **Help Documentation**: Represent use cases
- **Email Templates**: Header graphics

#### AI Assistant Avatar (Pilot)
- **Chat Interface**: Main chatbot representation
- **Notifications**: System messages and alerts
- **Tooltips**: Contextual help
- **Empty States**: Encouraging messages
- **Success Screens**: Celebrating achievements
- **Error Messages**: Friendly error handling
- **Onboarding Tutorial**: Guide character
- **Feature Announcements**: Introducing new features

- [ ] Appropriate lighting and shadows

- [ ] Professional appearance
- [ ] Appropriate diversity representation

- [ ] Appropriate for target audience
- [ ] Consistent with brand guidelines
