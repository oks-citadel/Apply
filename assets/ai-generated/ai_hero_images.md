# AI Hero Image Specifications
## ApplyforUs Platform

---

## Overview

Hero images are critical visual elements that communicate brand values, features, and benefits at first glance. These specifications ensure consistency, quality, and effectiveness across all platform touchpoints.

---

## Technical Specifications

### Image Dimensions

#### Desktop
- **Large Desktop**: 2560x1440px (2K)
- **Standard Desktop**: 1920x1080px (Full HD)
- **Laptop**: 1440x900px

#### Mobile
- **Tablet**: 1024x768px
- **Mobile**: 750x1334px (vertical)
- **Mobile Hero**: 1242x2688px (vertical)

### File Formats
- **Primary**: WebP (web optimization)
- **Fallback**: JPG (quality 85-90)
- **Print**: PNG (high-res)
- **Source**: PSD/AI (editable layers)

### Optimization
- File size: < 300KB for web
- Progressive loading support
- Responsive variants
- Retina-ready (2x assets)

---

## Brand Visual Language

### Color Palette
- **Primary Gradient**: Purple (#8B5CF6) to Indigo (#4C1D95)
- **Secondary Gradient**: Blue (#3B82F6) to Cyan (#06B6D4)
- **Accent Colors**:
  - Success: Green (#10B981)
  - Warning: Orange (#F97316)
  - Highlight: Pink (#EC4899)

### Composition Rules
- **Rule of Thirds**: Key elements at intersection points
- **Visual Balance**: Asymmetric but balanced
- **Breathing Room**: 60-70% negative space
- **Focus Point**: Clear primary subject

### Lighting
- **Style**: Soft, contemporary, optimistic
- **Direction**: Top-right to bottom-left
- **Color Temperature**: Slightly warm (5500K-6500K)
- **Contrast**: Medium-high for clarity

## Hero Image Component System

### Modular Elements
Create separate assets that can be combined:

1. **Background Layers**
   - Base gradient
   - Pattern overlay
   - Light effects
   - Particle systems

2. **Character/Subject Layers**
   - Professional figures
   - Hands/gestures
   - Devices/props

3. **UI Element Layers**
   - Job cards
   - Notifications
   - Charts/graphs
   - Icons and badges

4. **Effect Layers**
   - Sparkles/particles
   - Glows and halos
   - Motion blur
   - Shadows and depth

### Benefits of Modular System
- Easy to update individual elements
- Create variations quickly
- Maintain consistency
- Reduce file sizes
- Enable A/B testing

## Responsive Adaptations

### Desktop to Mobile Strategy

1. **Refocus**: Crop to main subject
2. **Reframe**: Adjust composition for vertical
3. **Reduce**: Simplify floating elements
4. **Resize**: Ensure text/UI remains readable
5. **Reoptimize**: Compress for mobile bandwidth

### Breakpoint Specifications

```css
/* Hero image responsive handling */
.hero-image {
  /* Desktop (1920px+) */
  background-image: url('hero-desktop-2x.webp');

  /* Laptop (1440px) */
  @media (max-width: 1536px) {
    background-image: url('hero-laptop-2x.webp');
  }

  /* Tablet (1024px) */
  @media (max-width: 1280px) {
    background-image: url('hero-tablet-2x.webp');
  }

  /* Mobile (750px) */
  @media (max-width: 1024px) {
    background-image: url('hero-mobile-2x.webp');
  }
}
```

## A/B Testing Variations

Create multiple variants for testing:

### Version A: Human-Focused
- Emphasize people and emotion
- Larger subject size
- Less UI elements
- Warmer color temperature

### Version B: Tech-Focused
- Emphasize UI and technology
- Smaller subject size
- More UI elements and data
- Cooler color temperature

### Version C: Balanced
- Equal emphasis on human and tech
- Medium subject size
- Moderate UI elements
- Neutral color temperature

## Animation Potential

For future animated hero sections:

1. **Parallax Scrolling**: Different layers move at different speeds
2. **Particle Animation**: Floating elements with subtle motion
3. **Fade-In Sequence**: Elements appear progressively
4. **Hover Effects**: Interactive element highlights
5. **Video Background**: Subtle looping movement
6. **Lottie Animations**: SVG-based lightweight animations

## Accessibility Considerations

- Maintain 4.5:1 contrast ratio for text overlays
- Provide alternative text descriptions
- Ensure images don't convey critical information alone
- Consider motion-reduced alternatives
- Test with screen readers
- Provide high contrast alternatives
