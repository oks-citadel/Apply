## File Naming Conventions

### Images
```
[type]_[category]_[description]_[variant]_[size]_[date].[ext]

Example:
hero_homepage_professional-working_v2_2560_20251208.webp
```

### Videos
```
video_[type]_[duration]_[platform]_[version]_[date].mp4

Example:
video_promo_30s_instagram_v2_20251208.mp4
```

### Animations
```
motion_[type]_[name]_[version]_[date].[ext]

Example:
motion_logo_fullintro_v2_20251208.json
```

### Voiceovers
```
vo_[project]_[language]_[version]_[voice]_[date].[ext]

Example:
vo_explainer_en-us_v1_femaleA_20251208.wav
```

## Quality Standards

### Technical Quality
- [ ] Appropriate resolution for use case
- [ ] Optimized file size
- [ ] Correct dimensions
- [ ] Proper color profile (sRGB for web)
- [ ] No artifacts or distortions

### Brand Consistency
- [ ] Uses approved brand colors
- [ ] Matches brand style
- [ ] Appropriate tone and mood
- [ ] Consistent with existing assets

### Accessibility
- [ ] Sufficient color contrast (4.5:1 minimum)
- [ ] Alt text provided
- [ ] Respects reduced motion preferences
- [ ] Readable text overlays

### Legal/Licensing
- [ ] Commercial use rights verified
- [ ] AI tool terms of service followed
- [ ] No copyright violations
- [ ] Proper attribution (if required)

## FAQ

**Q: Which AI tool should I use for [specific asset]?**
A: Check the "AI Tools Reference" section above for recommendations by asset type.

**Q: How do I ensure brand consistency?**
A: Always include exact hex codes in prompts, follow style guides in each document, and use the quality checklists.

**Q: What if the AI generates something I don't like?**
A: Iterate! Modify your prompt with specific improvements. Aim for 3-5 iterations. See troubleshooting in Asset Generation Workflow.

**Q: How do I handle licensing?**
A: Document which tool was used, ensure you have commercial rights (included in most paid AI tool subscriptions), and keep records.

**Q: Can I use these assets commercially?**
A: Yes, assets generated via paid Midjourney, DALL-E, and similar tools typically include commercial rights. Always verify in tool's terms of service.

**Q: How do I optimize file sizes without losing quality?**
A: Use WebP format, compress with TinyPNG or Squoosh, and create appropriately-sized variants. See Phase 5 in Asset Generation Workflow.

**Q: What resolution should I generate at?**
A: Generate at the highest resolution you'll need (typically 2560x1440px for hero images), then scale down for other uses.

**Q: How do I maintain consistency across a series of assets?**
A: Use the same prompts with minor variations, save successful prompts as templates, and use seed numbers when available.
