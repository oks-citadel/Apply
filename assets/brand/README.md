## Brand Guidelines Summary

### Logo Usage
- Always use official files (never recreate)
- Maintain aspect ratio
- Respect clear space requirements
- Use appropriate version for background color
- Minimum sizes: 120px digital / 0.75" print

[Full logo guidelines →](./logos/logo_usage.md)

### Color Usage
- Use semantic color naming
- Maintain WCAG AA contrast ratios (4.5:1 minimum)
- Test on both light and dark backgrounds
- Never use color alone to convey meaning
- Implement dark mode variants

[Full color system →](./color_swatches.md)

### Typography
- Headings: Inter Bold (36-80px)
- Body: Inter Regular/Medium (16-20px)
- UI: Inter Medium/SemiBold (14-16px)
- Line height: 1.5 for body, 1.2 for headings
- Letter spacing: -1% to -2% for large text

### Imagery
- Professional quality photos
- Diverse, inclusive representation
- Follow illustration style guide
- Use brand colors and patterns
- Optimize for web (compress images)

[Illustration guide →](./illustrations/illustration_style.md)

## Common Use Cases

### Website Header
- Use horizontal logo, 120-140px wide
- Place top-left corner with 20-40px padding
- Use appropriate color version (light/dark mode)
- Ensure good contrast with background

### Social Media Post
- Use square format (1080×1080px) for Instagram/Facebook
- Logo in bottom-right corner, 80-100px wide
- Maintain 40px padding from edges
- Use brand gradient or solid background

[Social media templates →](./social/social_templates.md)

### Email Signature
- Horizontal logo, 150px wide
- PNG format with transparent background
- Link to website
- Include alt text

### Presentation
- Title slide: Large centered logo
- Content slides: Small logo in corner (60-80px)
- Use brand colors and patterns
- High contrast for readability

[Mockup guide →](./mockups/mockup_guide.md)

## File Formats

### Vector Formats (Logos, Icons)
- **SVG**: Web use (preferred)
- **AI**: Adobe Illustrator source files
- **EPS**: Print/professional use
- **PDF**: Universal vector format

### Raster Formats (Photos, Mockups)
- **PNG**: Transparent backgrounds, web use
- **JPG**: Photos, solid backgrounds
- **WebP**: Modern web format (smaller files)

### Export Resolutions
- **@1x**: Standard resolution (base size)
- **@2x**: Retina displays (2× base size)
- **@3x**: High-DPI displays (3× base size)
- **Print**: 300 DPI minimum

## Brand Consistency Checklist

Before publishing any branded material, verify:

### Visual Elements
- ✓ Logo is official file (not recreated)
- ✓ Logo maintains correct proportions
- ✓ Clear space is respected
- ✓ Brand colors are used correctly
- ✓ Typography follows guidelines

### Accessibility
- ✓ Text contrast meets WCAG AA (4.5:1)
- ✓ Color is not sole indicator of meaning
- ✓ Alt text provided for images
- ✓ Readable at actual size

### Technical
- ✓ Correct file format for medium
- ✓ Proper resolution/DPI
- ✓ File size optimized
- ✓ Links work correctly
- ✓ Cross-browser/device tested

### Content
- ✓ Message is clear and on-brand
- ✓ No typos or grammar errors
- ✓ Complies with brand voice
- ✓ Legal/copyright verified
